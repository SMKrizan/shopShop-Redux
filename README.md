# shopShop-Redux

## Description
A MERN e-commerce platform that uses Redux to manage global state and Stripe to handle payments.

### Programming Languages, Platforms and Technologies Used:
- Apollo server
- CSS
- HTML
- ES6
- GraphQL
- Javascript
- JSX
- MERN stack:
-- MongoDB
-- Express
-- React
-- Node.js
- Redux
- Stripe
